# 🌀 나루토 그림자 분신술 (Naruto Shadow Clone)

웹캠, 핸드 트래킹, 그리고 사용자 정의 학습 모델을 사용하여 실시간으로 나루토의 **그림자 분신술** 효과를 재현하는 브라우저 기반 애플리케이션입니다.

> **⚠️ 중요 사항:**
> - 학습된 수인(Hand sign) 데이터셋과 모델 파일은 이 저장소에 **포함되어 있지 않습니다.** `trainer.html`을 사용하여 직접 모델을 학습시켜야 합니다.
> - **브라우저**: Chrome 권장 (Safari에서는 오작동할 수 있음)
> - **장치**: 웹캠이 반드시 필요합니다.

## 📝 참고 사항

- Python이 아닌 JavaScript로 구현된 머신러닝(ML) 프로젝트입니다.
- 전문적인 데이터 과학자가 아닌, 기능을 실제로 구현하는 데 목적을 두고 제작되었습니다. 최적화는 여러분의 몫입니다!
- 기본 버전에서는 단순한 신경망(Neural Network)을 사용했습니다(`trainer.js` 참고). 더 나은 성능을 원하신다면 모델 구조를 변경하거나 새로운 모델을 시도해 보시길 권장합니다.

## 📁 파일 구성

| 파일명            | 설명                               |

## ⚙️ 작동 원리

1. **MediaPipe Holistic**: 웹캠을 통해 손의 마디(Landmarks)를 실시간 추적합니다.
2. **MediaPipe Selfie Segmentation**: 배경에서 사용자의 신체만 분리합니다.
3. **TensorFlow.js 모델**: 사용자가 직접 학습시킨 신경망이 특정 양손 제스처를 인식합니다.
4. **효과 발동**: 제스처가 높은 신뢰도로 감지되면 연기 효과와 함께 그림자 분신이 나타납니다.

## 🚀 시작하기

### 1. 로컬 서버 실행

1. 웹캠 사용을 위해 로컬 서버가 필요하므로 [Node.js](https://nodejs.org/en)가 설치되어 있어야 합니다.
2. 저장소 루트 폴더에서 다음 명령어를 입력합니다:  
   `npx serve -p 3000` (3000번 포트 기준)
3. 브라우저에서 다음 주소로 접속합니다:
    - 트레이너 페이지: `http://localhost:3000/trainer`
    - 메인 페이지: `http://localhost:3000/index` (또는 `http://localhost:3000`)

### 2. 제스처 모델 학습

1. Chrome에서 트레이너 페이지에 접속합니다.
2. 원하는 수인(양손이 모두 보이게) 샘플을 녹화합니다.
3. 부정 샘플(무작위 손 동작 등)도 함께 녹화하여 정확도를 높입니다.
4. **Train** 버튼을 클릭하면 `gesture-model.json`과 `gesture-model.weights.bin` 파일이 생성됩니다.
5. 다운로드된 두 파일을 프로젝트 루트 폴더(메인 앱과 같은 위치)에 넣습니다.

### 3. 앱 실행

1. 메인 페이지(`index.html`)에 접속합니다.
2. 카메라 접근 권한을 허용합니다.
3. 학습시킨 수인을 취하면 화면에 그림자 분신이 나타납니다!

## 🛠 커스터마이징

- **`script.js`**: `customClones` 배열에서 분신의 위치, 크기, 지연 시간을 조절하거나 `predictGesture` 함수에서 인식 임계값(기본 0.999)을 수정할 수 있습니다.
- **`trainer.js`**: 모델 구조(Topology)나 학습 과정을 최적화할 수 있습니다.
